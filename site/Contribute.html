Contribute

We would love to have feedback from others downloading and installing Melati.

If you would like to contribute to the melati code, please join the development mailing list .

Please review the RoadMap for things that need to be done. More immediate tasks can be found on SourceForge.

In principle we give CVS write access to anyone who asks. The only quality policy we ask people to enforce is that committed changes should compile correctly, and that they should be believed to work. To make changes, please follow this procedure:

  1. Submit a Bug Report or Feature Request detailing what you plan to do/fix.
  2. Make changes and commit them to CVS.
  3. Submit a Change Notice detailing what you have done.
