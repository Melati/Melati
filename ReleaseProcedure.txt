Hello,

Melati is built using Maven. 

HOW TO USE

To build a distributable copy of Melati just type:

../doit


MAKING A RELEASE

Once you have the scripts working there are a few things you should do
to create a release:

1.  Change the version number and date in ../pom.xml
1a. Remove SNAPSHOT from currentVersion in ../pom.xml 
    and increment it. 
2.  If your change requires upgrade action by the user make a note
    in the Upgrading section of ../src/site/resources/README.html
3.  Change Last Updated and Updated By and add a new paragraph to 
    ../src/site/resources/RELEASE-NOTES.html
4.  Add a paragraph about this release to ../src/site/xdoc/Releases.xml
    Update ../src/site/xdoc/Download.ahtml
    Review ../src/site/xdoc/Roadmap.ahtml
5.  CVS commit all of the above
6.  CVS tag this release (cvs tag Melati-0-7-8)
9.  Test everything
10. 
11. On rama:
    cd /dist/melati
    cvs update
    mvn install
12. 
13. Update sourceforge
cvs -z3 -d:ext:developername@cvs.sourceforge.net:/cvsroot/melati co melati
cvs -d:ext:developername@melati.org:/usr/cvsroot export melati
cvs commit
