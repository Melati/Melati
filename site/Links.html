Links

Melati related Technologies:

  • Melati on SourceForge - resources for development and downloading Melati.
  • Paneris is a community of web developers, with a wealth of example code to share.
  • Webmacro is a template engine used in Melati.
  • Velocity another template engine supported by Melati.
  • Postgres is the reference database for initial Melati development.
  • MySQL has recently added transaction handling and is now supported by Melati.
  • MckoiDB a pure java database supported by Melati.
  • Linux is the reference operating system for initial Melati development.
  • Apache is the reference web server system for initial Melati development.
  • Jserv was the reference servlet runner for initial Melati development.
  • Tomcat is a more modern servlet runner, to which Melati and most Melati-driven sites are migrating.
  • Jetty is another Servlet 2.2 servlet runner.
  • CVS Concurrent Version System - you need a CVS client to download Melati.
  • WinCVS Concurrent Version System for Windows.
