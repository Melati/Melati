Example

Melati builds a basic database 'out of the box'. This is what you will get when you have completed the installation procedure, check it out. Please feel free to play about as much as you like. If you attempt to make changes to restricted data, you will be challenged for a LoginId / Password. To continue, use the ID of _administrator_ and password FIXME.

Included in the distribution is a simple example application. check it out.

If you have any problems, please join the melati users mailing list or email me.

More sophisticated examples will be posted shortly, these will demonstrate implementations of Messageboard and Shopping Trolley.
