Release Status (Version 0.7.0)

Melati is stable Beta software. Please subscribe to the announcements mailing list.

  • Support for the JSDK2.0, JSDK2.2 and JSDK2.3 APIs.
  • Support for Velocity 1.2 and 1.3RC1
  • Support for Webmacro 1.0
  • Support for Postgresql (version 6 to current)
  • Support for MySQL (only recent versions)
  • Support for HSQLDB, which is included.
  • Support for MckoiDB
  • Melati is available as a zip, tar or war download.

Melati is currently being used for several substantial commercial projects .
See the roadmap to a full release (1.0).
