Example Application

Included in the distribution is a simple Contacts Database application allowing addition, modification, deletion and search of Contact records.

The application runs on an SQL database, which is created from the DSD (Data Structure Definition). The created database is immediately usable through the generic Admin Interface.

See Search.java or just browse the CVS node
